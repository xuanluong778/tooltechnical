# Admin SaaS Metrics

Tài liệu này mô tả các chỉ số SaaS cần thêm vào trang Admin DigiSEO.

---

## 1. Hiện trạng

Admin hiện đã có:

```txt
Quản lý user
Đổi role/status
Bật/tắt API access
Bật/tắt API Admin pool
Xem dữ liệu user
Audit logs
```

Cần thêm chỉ số vận hành SaaS.
